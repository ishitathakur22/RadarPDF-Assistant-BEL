
### Yeh bahut simple hai — sirf `ROOT_PATH` variable change karна hai

`streamlit_app.py` mein yeh line hai:

```python
ROOT_PATH = "./sample_pdfs"
```

Yeh abhi tumhare **test folder** ko point kar rahi hai. BEL machine pe, isे us **actual folder ke path** se replace karना hoगा jahан unki saari radar-related PDFs stored hain.

### Karna kya hai:

**Step 1** — BEL machine pe pata karो unki files kahan store hain. Terminal se explore karो:

```bash
ls ~/
ls /home/<username>/
```

Ya agar files kisी shared drive/network folder mein hain:

```bash
ls /mnt/
ls /media/
```

Jab tak exact path na mile jahан unki PDFs hain (jaise `/home/bel_user/Documents/RadarDocs/` ya kuch aisा), isे note kar lो.

**Step 2** — `streamlit_app.py` mein `ROOT_PATH` update karो:

```python
ROOT_PATH = "/home/bel_user/Documents/RadarDocs"
```

(Apna exact path daalो, jo Step 1 mein mila)

**Step 3** — App restart karो (ya systemd service restart karो):

```bash
sudo systemctl restart radarpdf
```

---

### Zaroori baат — pehli baar bahut saari PDFs honi ki wajah se:

Agar BEL ke system mein **already sainkड़ों PDFs** hain (jaisा tumने shuru mein bataya "bahut saari files"), toh **pehli baar jab app is naye `ROOT_PATH` ko scan karेगी, poora indexing hoगी** — jo **kaafi zyada time** le sakती hai (jaisा humने dekha, ek 300-page PDF mein hi 5-8 minute lagे the — agar sainkड़ों PDFs hain, toh yeh **kई ghante** bhi lag sakते hain, ek baar ki cheez).

**Recommendation** — pehli baar `ROOT_PATH` set karके app chalाने ke baad, usे **bahut der (jaise raात bhar, ya kई ghante)** idle chhod dो, taaki poora initial indexing complete ho jaaye. Uske baad, sirf naye PDFs hi process honge (fast), jaisा humने practice kiya.

---

### Agar files **network drive / shared folder** mein hain (na ki local disk pe):

Us case mein pehle us drive ko **mount** karна padेगा Linux system mein (jaise NFS ya SMB share), phir uska mounted path `ROOT_PATH` mein daालना hoगा. Yeh setup network-specific hoगा — agar aisा case hai, batाओ, alag steps denge.

---

Pehle confirm karो — BEL machine pe files **kis exact folder mein** hain (ya honi chahiye), phir uske hisaab se `ROOT_PATH` set kar denge.
