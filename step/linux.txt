Chalo poora, shuru se aakhir tak, RHEL 9.2 ke liye complete guide — pen drive se lekar app chalne tak.

---

## Step 1 — Pen drive se project copy karna

```bash
ls /media/$USER/
```

Pen drive ka naam pata karके:

```bash
mkdir -p ~/BEL
cp -r /media/$USER/<PENDRIVE_NAME>/PDF_QA_System ~/BEL/
cd ~/BEL/PDF_QA_System
ls
```

---

## Step 2 — Python 3.11 install karo (RHEL 9.2 specific)

```bash
python3 --version
```

RHEL 9 mein default Python 3.9 hoता hai. 3.11 install karo:

```bash
sudo dnf install python3.11 python3.11-devel -y
python3.11 --version
```

Agar yeh error de (package na mile), EPEL enable karo:

```bash
sudo dnf install epel-release -y
sudo dnf install python3.11 python3.11-devel -y
```

---

## Step 3 — Baaki system packages install karo

```bash
sudo dnf update -y
sudo dnf install tesseract ffmpeg -y
```

Agar `tesseract` na mile, EPEL se try karo (upar wala EPEL command pehle chala lena).

## Step 7 — Ollama install karo (kyunki nahi hai)

Pehle **Windows machine pe (internet ke sath)** yeh karo:

1. Browser mein kholो: `https://ollama.com/download/ollama-linux-amd64.tgz`
2. File download karके pen drive mein copy karो
3. Windows pe `C:\Users\enggm\.ollama\models` poora folder bhi pen drive mein copy karो

Ab **RHEL machine pe**:

```bash
cp /media/$USER/<PENDRIVE_NAME>/ollama-linux-amd64.tgz ~/
cd ~
tar -xzf ollama-linux-amd64.tgz
sudo mv bin/ollama /usr/local/bin/ollama
sudo chmod +x /usr/local/bin/ollama
ollama --version
```

Model files copy karo:

```bash
mkdir -p ~/.ollama/models
cp -r /media/$USER/<PENDRIVE_NAME>/.ollama/models/* ~/.ollama/models/
```

Test karo:

```bash
ollama serve &
ollama list
```

`llama3.2` dikhна chahiye. Test karके dekho:

```bash
ollama run llama3.2
```

Kuch poochके `/bye` se exit karो.

**Architecture confirm karна na bhूlो:**

```bash
uname -m
```

Agar `x86_64` na aye, batао — alag binary chahiye hoगी.

## Step 8 — ROOT_PATH set karna (tumhari actual PDFs kahан hain)

Pehle dhundo BEL ki files kahан hain:

```bash
ls ~/
find / -iname "*.pdf" 2>/dev/null | head -20
```

Yeh command system mein PDFs dhundeगa aur top 20 dikhaएगा — usse pata chal jaayega woh kis folder mein hain (jaise `/home/user/Documents/RadarFiles/`).

Ya agar exact folder pata hai, seedha check karो:

```bash
ls /path/to/that/folder
```

Jab exact path mil jaaye (jaise `/home/beluser/RadarDocs`), `streamlit_app.py` file kholो:

```bash
nano streamlit_app.py
```

`Ctrl+W` dabाके search karो `ROOT_PATH`, yeh line milेगी:

```python
ROOT_PATH = "./sample_pdfs"
```

Isे replace karो apne exact path se:

```python
ROOT_PATH = "/home/beluser/RadarDocs"
```

Save karo (`Ctrl+O`, Enter, `Ctrl+X`).

**Purana index delete karके fresh start karo** (kyunki purana test data ka index hai):

```bash
rm -f saved_index/faiss_index.bin
rm -f saved_index/chunks.pkl
```

## Step 11 — Auto-start systemd services

**Ollama service:**

```bash
sudo nano /etc/systemd/system/ollama.service
```

```ini
[Unit]
Description=Ollama Service
After=network.target

[Service]
ExecStart=/usr/local/bin/ollama serve
User=<tumhara-username>
Restart=always
Environment="HOME=/home/<tumhara-username>"

[Install]
WantedBy=multi-user.target
```

**Streamlit service:**

```bash
sudo nano /etc/systemd/system/radarpdf.service
```

```ini
[Unit]
Description=RadarPDF Assistant
After=network.target ollama.service

[Service]
ExecStart=/home/<username>/BEL/PDF_QA_System/pdf_qa/bin/python3.11 -m streamlit run /home/<username>/BEL/PDF_QA_System/streamlit_app.py --server.port=8501 --server.address=0.0.0.0 --server.fileWatcherType none
WorkingDirectory=/home/<username>/BEL/PDF_QA_System
User=<username>
Restart=always
Environment="HOME=/home/<username>"

[Install]
WantedBy=multi-user.target
```

Enable + start:

```bash
sudo systemctl daemon-reload
sudo systemctl enable ollama radarpdf
sudo systemctl start ollama radarpdf
sudo systemctl status ollama radarpdf
```

Agar SELinux permission error aaye (RHEL mein common), batао — alag se handle karenge.

## Step 12 — Reboot test karo


Login karo, wait karो ~15-20 second, browser mein `http://localhost:8501` — bina kuch chalाए app khुl jaana chahiye.
